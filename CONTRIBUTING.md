# Contributing Guidelines

Thank you for helping future Embedded Systems students.

## How to contribute

1. Fork this repository
2. Create the appropriate folder if it does not exist
3. Upload clear, readable PDFs only
4. Follow this naming format:
   COURSECODE_EXAMTYPE_YEAR.pdf

Example:
MAEDS503_CAT1_2025.pdf

5. Create a Pull Request with a short description

## Rules
- No handwritten solutions
- No copyrighted books or notes
- Only question papers
- Do not upload irrelevant material
