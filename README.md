# M.Tech Embedded Systems Question Papers

This repository contains **M.Tech Embedded Systems question papers**
(CATs and FATs) collected from **VIT Vellore**.

The goal of this repository is to help students prepare for exams when
previous year question papers are not officially available.

## Courses Covered

- MAEDS501 – Embedded System Design
- MAEDS502 – Microcontroller Architecture and Organization
- MAEDS503 – Embedded Programming
- MAEDS504 – In Vehicle Networking
- MAEDS506 – System Design Using FPGA

## Exam Types
- CAT 1
- CAT 2
- FAT (Final Assessment Test)

## Who can contribute?
Any student or alumnus who has **legally obtained question papers**
can contribute by uploading scanned PDFs.
